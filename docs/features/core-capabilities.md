# Core Capabilities

## Summary
LurkAssistant provides foundational capabilities that extend beyond its primary user interface.

## Value Provided
- Enables automation within the LurkAssistant domain.
- Provides shared logic and utilities.


## Newly Documented Existing Capabilities
- `--help` / `-h` → prints all available options
- `--streamer <name>` → overrides the lurk target using a single streamer name
- `--delay <seconds>` → overrides `IterationDelay`
